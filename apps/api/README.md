# Orgx API

This is the `FastAPI` backend for Orgx.

It is responsible for:

- public SaaS onboarding and tenant provisioning
- tenant-scoped user and employee management
- attendance and remote work-proof validation
- payroll, approvals, and payouts
- audit logging and record verification

Current scaffolded routes:

- `/health`
- `/api/v1/public/plans`
- `/api/v1/public/tenants/resolve`
